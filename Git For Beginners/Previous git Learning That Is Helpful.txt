Set Up Git: 
git config --global user.name "Your Name"
 git config --global user.email "youremail@domain.com"

git branch - to list all the Git local branches
git branch (your desired branch name) - To create new branch
git switch master - switch to the “master” branch from another local branch
git checkout master-Another way to switch from any local branch to a “master” branch is the “git checkout” command
 git clone (url) 
 git add (file names) -for specific file
git add -A - for all files inside a folder
 git commit -m "(committed message) -for commit whole folder
git commit -m "(committed message) your file - for commit specific file
